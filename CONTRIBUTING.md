# Contributing

Thanks for helping improve MCP Security Toolkit.

## Good First Contributions

- Add a new risky tool name rule.
- Add support for a new MCP client config format.
- Add safe or unsafe example configs.
- Improve README examples.
- Add tests for scanner rules.

## Development

```bash
npm install
npm test
npm run check
```

## Pull Request Checklist

- [ ] Tests pass with `npm test`.
- [ ] New scanner behavior includes tests.
- [ ] New examples do not include real secrets.
- [ ] Docs are updated when behavior changes.

## Security Reports

Please do not open public issues for sensitive vulnerabilities. See [SECURITY.md](SECURITY.md).
